What is this website about?

The main question here is:
What purpose does this site fulfil?
The answer to this is simple: none. However, this website represent my chance to learn some HTML and CSS, plus: why not?

Why no JS?

A lot of websites rely on Javascript for a few years now (tendency rising).
This not only is a security and privacy risk,
it also really limits, what you can do with a text-only browser like
Lynx, w3m or even graphical ones like Netsurf.
Besides that, I want to learn how to provide a website that:
  • Looks rather cool, even in modern browser
  • Is fully functional in any browser

Something else?

I want to provide and develop this website as openly as possible.
I currently use Apache on my own Debian-Server to host this website.
You can also find the entire code and assets used on here on my selfhosted Git-Service.
I currently have no bugtracker, but you can E-Mail me to talk about issues and discover how to contribute.
Unfortunately, this website is published using NGrok, which is proprietary software.
